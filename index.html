Timeshift备份
发表于2023-02-10
timeshift有两种备份方式： 1.rsync，有点像复制粘贴备份，比较无脑 2.btrfs文件系统的一个特性，启用该备份方式必须在Linux安装阶段就开始配置，不然比较麻烦。 首先创建btrfs文件系统后，挂载分区并进入分区的根目录创建子卷： btrfs subvolume create ./@ #挂载点为/ home分区同上： btrfs subvolume create ./@home #挂载点为/home chroot时需挂载子卷，具体原因见FHS（LInux文件目录标准）： mount -t btrfs -o subvol=/@ ...mount -t btrfs -o subvol=/@home ... 同样，/etc/fstab也需挂载子卷。安装好系统后再配置时需要重新配置grub，并移动大量的文件，这就是本文开始时说明的麻烦的地方。
